Music Magazine

Music Magazine is a web application created for the course Programming 3. The goal of the assignment was to load and display data from a JSON file using JavaScript. I chose music as the theme of my application, resulting in an interactive mini-magazine filled with albums from various artists and genres.

Each album includes detailed information and offers a short audio preview so users can quickly get a feel for the music. Additionally, users can mark albums as favorites, creating a personalized list of their preferred tracks and artists.

The project demonstrates my ability to work with JSON data, manipulate the DOM dynamically, and build a clean, interactive user interface using only HTML, CSS and JavaScript.
